# Matcha & Me — Landing Page Concept

A modern, editorial landing page concept inspired by **Matcha & Me**, Kolkata's first matcha café. Built as a frontend design and development exercise exploring brand storytelling, motion design, and immersive user experience.

> **Disclaimer:** This is an independent design concept created for educational and portfolio purposes. It is not affiliated with, commissioned by, or endorsed by Matcha & Me.

## About the Project

This project started as a question: *what would it feel like to walk into Kolkata's first matcha café before you even got there?*

The result is a landing page that leans into atmosphere — slow reveals, earthy tones, and editorial-style typography — to build anticipation and tell a sensory story. Every section was designed with intention, from the Flavor Journey to the visit details, to feel less like a website and more like a mood.

---

## Features

- **Responsive layout** — fluid across mobile, tablet, and desktop
- **Interactive Flavor Journey** — scroll-driven exploration of the matcha menu
- **Coffee showcase section** — visual storytelling for the full drink lineup
- **Editorial menu experience** — typography-forward design that lets the menu breathe
- **Smooth animations** — orchestrated reveals and micro-interactions via Framer Motion
- **Location & visit section** — designed to convert curiosity into a real visit
- **Mobile-first approach** — touch-friendly and fast on any screen size

## Getting Started

**Prerequisites:** Node.js v18+ and npm

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Design Decisions

- **Color palette** drawn from matcha's natural tones — deep greens, warm creams, and earthy neutrals
- **Typography** chosen to feel editorial and calm, pairing a display serif with a clean sans-serif body
- **Motion** kept intentional and restrained — entry animations and hover states, not spectacle
- **Layout** inspired by Japanese minimalism and modern café branding
